# DESAFIO-QA-BEEDOO-2026
## Qual você acredita ser o objetivo da aplicação?
A aplicação tem como objetivo permitir o cadastro e gerenciamento de cursos, possibilitando que os usuários criem cursos e visualizem os mesmos cadastrados na plataforma.
## Quais são os principais fluxos disponiveís?
Os principais fluxos disponiveis são:
- Cadastro de cursos
- Visualização da listagem de cursos
- Edição das informações do curso disponivel
- Navegação entre as telas de aplicação
## Quais pontos do sistema você considera mais críticos para teste?
Os pontos mais críticos para teste:
- Cadastros de novos cursos
- Validação dos campos obrigatórios
- Consistência dos dados (cadastro duplicado)
- Qualidade dos textos exibidos na interface
- Exibição correta das informações cadastradas
